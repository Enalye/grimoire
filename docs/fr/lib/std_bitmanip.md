# std.bitmanip

Opérations pour effectuer des opérations binaires.
## Opérateurs
|Opérateur|Entrée|Sortie|
|-|-|-|
|&|**int**, **int**|**int**|
|&|**uint**, **uint**|**uint**|
|&|**byte**, **byte**|**byte**|
|<<|**int**, **int**|**int**|
|<<|**uint**, **uint**|**uint**|
|<<|**byte**, **byte**|**byte**|
|>>|**int**, **int**|**int**|
|>>|**uint**, **uint**|**uint**|
|>>|**byte**, **byte**|**byte**|
|^|**byte**, **byte**|**byte**|
|^|**uint**, **uint**|**uint**|
|^|**byte**, **byte**|**byte**|
|\||**int**, **int**|**int**|
|\||**uint**, **uint**|**uint**|
|\||**byte**, **byte**|**byte**|
|~|**int**|**int**|
|~|**uint**|**uint**|
|~|**byte**|**byte**|
