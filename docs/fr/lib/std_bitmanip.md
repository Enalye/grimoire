# bitmanip

## Description

Opérations pour effectuer des opérations binaires

## Opérateurs

|Operateur|Entrée|Sortie|
|-|-|-|
|<<|**int** *valeur*, **int** *décalage*|**int**|
|>>|**int** *valeur*, **int** *décalage*|**int**|
|&|**int** *a*, **int** *b*|**int**|
|\||**int** *a*, **int** *b*|**int**|
|^|**int** *a*, **int** *b*|**int**|
|!|**int** *a*, **int** *b*|**int**|
