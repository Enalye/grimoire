## Function descriptions

<a id="func_0"></a>
> assert (*value*: **bool**)

If `value` is false, throw an exception `"AssertError"`.

<a id="func_1"></a>
> assert (*value*: **bool**, *error*: **pure string**)

If `value` is false, throw the exception.
